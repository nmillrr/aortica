This directory is where the INT8 ONNX edge model (model.onnx) and
model_metadata.json should be placed.

The model is generated by the Python export pipeline:

    from aortica.edge.mobile_export import export_mobile_model
    export_mobile_model("aortica_edge_int8.onnx", "mobile/android/app/src/main/assets/")

Or bundled automatically during CI/CD builds (see US-110).
